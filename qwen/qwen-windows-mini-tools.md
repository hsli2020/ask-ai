# 🛠️ 10+ 类似TinyTask的Windows小巧实用工具推荐

> TinyTask是一款仅36KB的极简自动化录制工具[[17]]，以下推荐的工具同样具备**体积小巧、便携免安装、功能实用**的特点：

---

## 🔁 自动化/宏录制类（最接近TinyTask）

| 工具名称 | 体积 | 主要功能 | 特点 |
|---------|------|---------|--------|
| **AutoHotkey** | ~2MB | 脚本自动化、热键定制、宏录制 | 开源强大，支持复杂脚本，社区活跃[[1]] |
| **GS Auto Clicker** | <1MB | 自动模拟鼠标点击 | 界面简洁，支持点击间隔/次数设置[[7]] |
| **TGMacro** | <500KB | 游戏/应用宏录制回放 | 轻量简单，适合游戏挂机[[1]] |
| **Perfect Automation (Portable)** | ~3MB | 录制鼠标键盘动作并回放 | 支持USB便携运行，配置保存[[19]] |

## 🔍 系统增强/效率工具类

| 工具名称 | 体积 | 主要功能 | 特点 |
|---------|------|---------|--------|
| **Everything** | ~2MB | 秒级文件/文件夹搜索 | 索引极速、资源占用极低、支持便携版[[68]][[69]] |
| **WizTree Portable** | ~4MB | 磁盘空间快速分析 | 读取MFT直接扫描，速度远超传统工具[[79]][[80]] |
| **Ditto Portable** | ~5MB | 高级剪贴板管理 | 保存历史剪贴内容、支持搜索/分类/同步[[107]][[112]] |
| **Volume²** | <2MB | 高级音量控制替代 | 支持鼠标滚轮/热键/屏幕边缘调节音量[[97]][[101]] |
| **QuickTextPaste** | <500KB | 快捷键快速插入预设文本 | 便携免装，支持多语言模板、命令执行[[89]][[90]] |

## 🧰 NirSoft工具集（超推荐！）

**NirLauncher** 是一个包含200+小巧工具的便携合集[[40]]，单个工具通常仅几十~几百KB，全部免安装：

| 推荐子工具 | 功能 | 体积 |
|-----------|------|------|
| **OpenedFilesView** | 查看被占用的文件/进程 | ~100KB |
| **WirelessKeyView** | 查看保存的WiFi密码 | ~150KB |
| **RegScanner** | 快速搜索注册表项 | ~80KB |
| **BlueScreenView** | 分析系统蓝屏日志 | ~200KB |
| **PasswordFox/ChromePass** | 查看浏览器保存的密码 | ~120KB |

> ⚠️ 部分密码恢复工具可能被杀软误报，属正常现象[[40]]

## ✅ 选择建议

- 🔹 **需要录制操作** → TinyTask / AutoHotkey / GS Auto Clicker
- 🔹 **提升日常效率** → Everything + Ditto + QuickTextPaste
- 🔹 **系统维护排查** → NirLauncher 合集 + WizTree
- 🔹 **追求极致小巧** → 优先选 <1MB 的工具（如QuickTextPaste、GetWindowText）
