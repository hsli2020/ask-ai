# Windows 11 运行 Android 应用的具体操作步骤

> ⚠️ **重要提示**：微软官方的 Windows Subsystem for Android (WSA) 已于 **2025年3月5日** 正式终止支持 [[30]][[32]]。但中国区用户可通过**腾讯应用宝专区**继续使用，或采用第三方方案运行已安装的子系统。

---

## 🔹 方案一：通过腾讯应用宝专区（中国区推荐，最简单）

这是微软与腾讯合作的官方方案，适合普通用户一键安装安卓应用 [[2]][[49]]。

### 前置条件
- Windows 11 系统已更新至最新版本
- Microsoft Store 版本为 **22410.1402.100.0 或更高** [[49]]
- 系统区域设置为「中国」

### 操作步骤
1. **打开 Microsoft Store** → 点击「库」→「获取更新」，确保商店为最新版
2. 在商店首页搜索或浏览找到 **「腾讯应用宝专区」** [[2]]
3. 进入专区后，选择需要的安卓应用（如抖音、快手等）
4. 点击「获取」→「安装」，应用会自动下载并部署
5. 安装完成后，应用图标会出现在**开始菜单和桌面**，双击即可运行 [[4]]

✅ 优点：操作简单、与系统集成度高、无需技术基础  
❌ 限制：应用库由腾讯应用宝提供，部分应用（如安卓版微信）可能暂不支持 [[2]]

## 🔹 方案二：使用已安装的 WSA 子系统（进阶用户）

如果您之前已安装 WSA，或希望使用第三方预构建包，可参考以下步骤 [[1]][[12]]。

### 前置检查
| 项目 | 要求 |
|------|------|
| 系统版本 | Windows 11 22000.257 或更高 |
| 内存 | 8GB 及以上（推荐 16GB） |
| 存储 | 系统盘为 SSD |
| 处理器 | Intel 8代/i3+、AMD Ryzen 3000+、或骁龙 8c+ |
| 虚拟化 | BIOS/UEFI 中启用虚拟化（Intel VT-x / AMD-V） |

### 步骤 1：启用 Windows 虚拟化功能
1. 按 `Win + R`，输入 `optionalfeatures` 回车
2. 勾选 ✅ **「虚拟机平台」** 和 ✅ **「Windows 虚拟机监控程序平台」**
3. 点击「确定」→ 重启电脑 [[5]][[12]]

### 步骤 2：安装 WSA 子系统（如尚未安装）
> 由于微软商店已下架，需通过第三方渠道获取预构建包（如 [WSABuilds](https://github.com/WSABuilds) 项目）[[27]][[38]]

1. 下载适用于您 CPU 架构的 `.msixbundle` 文件（x64 或 ARM64）
2. 以**管理员身份**打开 PowerShell，执行：
   ```powershell
   Add-AppxPackage -Path "D:\路径\到\文件\MicrosoftCorporationII.WindowsSubsystemForAndroid_xxx.msixbundle"
   ```
3. 安装完成后，开始菜单会出现「适用于 Android 的 Windows 子系统」[[1]]

### 步骤 3：配置并启动子系统
1. 打开「适用于 Android 的 Windows 子系统 设置」
2. 将「子系统资源」设为「连续」提升性能
3. 开启「开发人员模式」，记录显示的 IP 地址和端口（如 `127.0.0.1:58526`）[[8]][[13]]
4. 点击「文件」图标启动 Android 环境

### 步骤 4：安装 APK 应用（两种方式）

#### 方式 A：使用 WSATools（图形界面，推荐新手）
1. 在 Microsoft Store 搜索安装 **「WSATools」** [[1]]
2. 打开工具 → 点击「安装 APK」→ 选择本地 `.apk` 文件
3. 等待安装完成，应用会自动出现在开始菜单

#### 方式 B：使用 ADB 命令（适合开发者）
1. 下载 [Android SDK Platform-Tools](https://developer.android.com/studio/releases/platform-tools)
2. 解压后，在平台工具目录打开命令提示符
3. 执行以下命令：
   ```bash
   adb connect 127.0.0.1:58526          # 连接 WSA（端口以子系统设置中显示的为准）
   adb install 应用名称.apk             # 安装 APK
   ```
4. 安装成功后，应用即可在 Windows 中运行 [[13]][[38]]

## 🔹 常见问题解答

| 问题 | 解决方案 |
|------|----------|
| **找不到腾讯应用宝专区？** | 确保系统区域为中国、Microsoft Store 已更新至最新版 [[49]] |
| **WSA 启动失败/黑屏？** | 检查 BIOS 虚拟化是否开启；确认 Hyper-V 相关功能已启用 [[12]] |
| **安装 APK 时提示连接失败？** | 确认子系统已启动 + 开发人员模式已开启 + ADB 端口正确 [[13]] |
| **应用无法联网？** | 在子系统设置中检查网络权限；部分应用需手动授权网络访问 |
| **性能卡顿？** | 将子系统资源设为「连续」；关闭其他大型程序；确保使用 SSD |
