# 股票量化数据获取渠道指南（2026版）

做股票量化分析，数据源的选择至关重要。以下是按**免费/付费**、**国内/海外**分类的主流数据渠道推荐：

---

## 🆓 免费开源数据源（适合入门/研究）

### 1️⃣ AData（专注A股）
- **特点**：专注A股量化，多数据源融合切换，保证高可用性 [[7]]
- **安装**：`pip install adata`
- **优势**：
  - 支持日线/周线/月线K线、实时行情、资金流向
  - 内置代理设置功能，应对接口限流
  - 完全免费开源，适合个人量化需求
- **示例代码**：
  ```python
  import adata
  # 获取股票代码
  res_df = adata.stock.info.all_code()
  # 获取行情数据
  res_df = adata.stock.market.get_market(stock_code='000001', k_type=1)
  ```

### 2️⃣ AKShare（全能型爬虫聚合）
- **特点**：聚合新浪、腾讯、东方财富等多源数据，覆盖股票/基金/期货/宏观 [[24]]
- **安装**：`pip install akshare`
- **优势**：
  - 完全免费，无需注册
  - 另类数据丰富（宏观指标、产业数据、舆情等）
  - 社区活跃，持续更新
- **注意**：基于爬虫机制，稳定性依赖目标网站，建议加`sleep(3)`间隔请求 [[24]]

### 3️⃣ BaoStock（证券宝）
- **特点**：免费开源证券数据平台，无需注册，返回pandas DataFrame格式 [[28]]
- **优势**：
  - 提供A股历史行情、财务数据、指数数据
  - 接口简洁，适合初学者快速上手
  - 支持复权处理、停牌数据处理

### 4️⃣ Tushare Pro（积分制）
- **特点**：老牌财经数据接口，基础数据免费，高级数据需积分 [[1]]
- **获取方式**：官网注册获取token，新用户送120积分
- **优势**：
  - 数据清洗质量高，复权/财报等边界处理完善
  - 500积分（约50元）即可解锁大部分A股日线接口
- **注意**：分钟级数据、港美股需额外付费，有频控限制 [[1]]

## ⚠️ 重要提醒

1. **合规使用**：所有数据源请遵守其使用协议，避免高频爬虫导致IP封禁 [[24]]
2. **本地缓存**：历史数据建议拉取后存本地（如Parquet格式），减少重复请求 [[1]]
3. **数据校验**：关键策略建议多源交叉验证，避免单一数据源异常影响回测结果
4. **实盘谨慎**：免费数据源延迟/断供风险较高，实盘交易务必使用付费专业接口

## 🔧 快速开始示例（AKShare获取A股日线）

```python
import akshare as ak
import pandas as pd

# 获取平安银行历史日线数据
df = ak.stock_zh_a_hist(symbol="000001", period="daily", 
                        start_date="20230101", end_date="20241231")
print(df.head())

# 简单计算20日均线
df['ma20'] = df['close'].rolling(20).mean()
```
